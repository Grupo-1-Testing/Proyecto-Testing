## Uso de la aplicación

### Prerequisitos
* ruby v. 3.0.1

### Instalación y Ejecución
 1. **Instalación:** ejecutar `bundle install`
 2. **Ejecución:** ejecutar `rake run`
 3. **Testeo:** ejecutar `rake test`
 4. **Linter**: ejecutar `rubocop` o `rake lint`
